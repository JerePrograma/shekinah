# Informe SEO

## Implementado

- títulos únicos y descripciones por ruta;
- canonical absoluto;
- Open Graph y Twitter Cards;
- imagen social por contenido;
- sitemap e índice de sitemap;
- `robots.txt`;
- favicon y web manifest;
- datos estructurados `Organization`, `Article`, `Recipe` y breadcrumbs;
- redirecciones de slugs duplicados o técnicos;
- 404 con `noindex`.

## Dominio

El canonical previsto es `https://shekinah.pages.dev`. Puede reemplazarse sin tocar código configurando `SITE_URL` durante el build y actualizando `public/robots.txt` cuando exista dominio definitivo.

## Datos no inventados

No se agregaron teléfono, WhatsApp, redes sociales, horarios, precios, stock, autores personales ni testimonios. La organización usa nombre, correo y dirección únicamente porque figuran en contenido legal publicado.

## Pendientes

- Verificar la URL real después del primer despliegue.
- Definir dominio personalizado si existe una decisión comercial.
- Revisar títulos/descripciones con el propietario antes de una campaña.
- Sustituir fotografías editoriales por activos propios si se dispone de ellos.
