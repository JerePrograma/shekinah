# Nombre del proyecto

El nombre público, editorial y de marca es **Shekinah**.

Los identificadores técnicos permanecen en minúsculas cuando forman parte de una URL, un nombre de paquete o el aislamiento de infraestructura:

- paquete npm privado: `shekinah`;
- repositorio: `JerePrograma/shekinah`;
- proyecto Cloudflare Pages: `shekinah`;
- dominio técnico: `shekinah-7dl.pages.dev`;
- proyecto Docker Compose de la restauración existente: `shekinah-original-reference`.

Estos identificadores no se muestran como nombre de marca dentro del sitio. La cabecera, el pie, los títulos HTML y la configuración pública de WordPress deben mostrar **Shekinah**.
