# Reconstrucción visual

## Recuperado directamente

- Emblema circular con barco, hojas y tonos verde/dorado.
- Firmas horizontal y vertical de Shekinah.
- Imágenes de local, frascos, especias, romero, cocina y mapa.
- Lenguaje editorial: origen, viaje, alquimia natural, bienestar y alta cocina.
- Secciones conceptuales de Inicio, Nosotros, Blog, Recetas y Tienda.

## Inferido

- Sistema de espaciado y grilla responsive.
- Componentes de tarjetas, hero, breadcrumbs, navegación y footer.
- Jerarquía tipográfica con serif editorial para títulos y sans-serif del sistema para cuerpo.
- Uso de verde bosque, crema y dorado extraídos visualmente de la marca.

## Rediseñado

- Cabecera sticky y menú móvil accesible.
- Portada basada en la página Inicio en lugar de un listado de entradas.
- Índices automáticos desde colecciones de contenido.
- Tienda como catálogo informativo.
- Recetas con ingredientes estructurados y datos `Recipe` cuando la evidencia alcanza.
- Página 404 y avisos de limitaciones.

## No verificable

- Pixelado, medidas y breakpoints exactos del tema anterior.
- Tipografías originales.
- Animaciones del constructor.
- Estado visual de cada pantalla y dispositivo.
- Orden exacto de todos los elementos remotos no disponibles.

## Código descartado

No se copió PHP, JavaScript del asistente, CSS completo del tema ni componentes propietarios. Los archivos del tema se usaron solo como señal analítica de estructura y fueron eliminados del entorno efímero.
